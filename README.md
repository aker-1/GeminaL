# What is GeminaL?

GeminaL is an open-source tool written in Bash, allowing to you <br>
to use ai in your terminal

<br><br>


# Clone Geminal Tool

`git clone https://github.com/aker-1/GeminaL.git`<br><br><br>


# Usage
`cd GeminaL`<br><br>
`chmod 777 *`<br><br>
`./GeminaL.sh`<br><br>
