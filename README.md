# What is GeminaL?

KasperScan is an open-source tool written in Bash, utilizing the Kaspersky API to scan :
- Hashes
- URLs
- IP Addresses
- Files
- And more...

<br><br>


# Clone Geminal Tool

`git clone https://github.com/aker-1/GeminaL.git`<br><br><br>


# Usage
`cd GeminaL`<br><br>
`chmod 777 *`<br><br>
`./GeminaL.sh`<br><br>
